Uses

Tools I use for web development

As a Front-end Dev, I use a lot of tools and extensions for better developer experience and productivity.😎

Most of the time I enable only those which I need at a given moment.



Today I want to share all tools I use for web development💁🏻‍♂️



💻 Terminal

Tabby is an infinitely customizable cross-platform terminal app.

oh-my-zsh is my default shell.

Z for quick cd.


⌨️ Code Editor

VSCode - An open source code editor from microsoft.

Cobalt2 Theme - My current theme now.


🔧 Editor Extensions

GitLens — Git supercharged - Supercharges the Git capabilities built into Visual Studio Code.

Markdown Preview Enhanced - Markdown Preview Enhanced ported to VSCode.

Color Highlight - Highlights color.

Auto Close Tag - Automatically add HTML/XML close tag, same as Visual Studio IDE or Sublime Text.

Auto Rename Tag - Auto rename paired HTML/XML tag.

Highlight Matching Tag - Highlights matching closing and opening tags.

Project Manager - Easily switch between projects.

Bracket Pair Colorizer - A customizable extension for colorizing matching brackets.

Prettier - VS Code plugin for Code formating.

Formatting Toggle - Allows you to toggle the Formatter (Prettier, Beautify, …) on and off with a simple click.

Cloak - Cloak hides/shows your secrets in environment files, to avoid accidentally sharing them with everyone who sees your screen.


🎉 Browsers

Brave (my default browser)

Chrome

Firefox Developer Edition


🔩 Chrome Extensions

React Developer Tools - A React debugging tools to the Chrome Developer Tools.

Redux DevTools - Redux DevTools for debugging application’s state changes.

Chrome Notes - A simple notepad extension for Google Chrome

Grammarly - Grammarly for Chrome offers real-time suggestions to help you write your best online, no matter what you’re working on in your browser.

ColorZilla - With ColorZilla you can get a color reading from any point in your browser, quickly adjust this color and paste it into another program.

JSON Viewer - It is a Chrome extension for printing JSON and JSONP in a beautiful format.

Internet Connection Monitor - Extension monitors internet connection status. Can detect when the internet doesn’t work over operating Wi-Fi / Ethernet.

Console Importer - Import JavaScript and CSS resources from console, with one command


👨🏻‍💻 Desktop tools

Postman - Postman Simplifies API Development & Testing.

AnyDesk - Connect to a computer remotely, be it from the other end of the office or halfway around the world using anydesk.


📊 Gnome extensions

system-monitor - Display system information in gnome-shell status bar, such as memory usage, cpu usage, network rates.


💯 Productivity

Trello - Trello helps teams move work forward.

Slack - Slack makes it downright pleasant to work together
